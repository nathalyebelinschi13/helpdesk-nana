# 🖥️ HelpDesk Nana System

Sistema de chamados (Help Desk) desenvolvido com HTML, CSS e JavaScript.

## 🚀 Funcionalidades
- Criação de chamados
- Controle de status (aberto/fechado)
- Definição de prioridade
- Persistência de dados com localStorage

## 🛠️ Tecnologias
- HTML
- CSS
- JavaScript

## 📌 Objetivo
Projeto desenvolvido para praticar lógica de programação, manipulação de DOM e simulação de sistema real de suporte técnico.
